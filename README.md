# 💻 Rada Samorządu Studentów MFI UG 

Cześć! 👋 Witamy na oficjalnym profilu GitHub Rady Samorządu Studentów Wydziału Matematyki, Fizyki i Informatyki Uniwersytetu Gdańskiego. 

Jako studenci nauk ścisłych lubimy tworzyć rzeczy od zera (i mieć nad nimi pełną kontrolę!). Ten profil powstał jako baza dla naszych projektów cyfrowych – przede wszystkim do tworzenia, hostowania i rozwijania stron internetowych naszych największych wydarzeń wydziałowych.

## 🚀 O co tutaj chodzi?
Budujemy społeczność „coMFI” – chcemy, żeby nasz Wydział był miejscem pełnym inspiracji, nauki i integracji. W tym repozytorium znajdziesz kody źródłowe stron, które promują nasze inicjatywy. Dzięki temu:
* Mamy pełną niezależność w projektowaniu naszych kampanii promocyjnych.
* Integrujemy studentów informatyki, dając im pole do popisu przy realnych projektach.
* Archiwizujemy historię naszych największych wydarzeń.

## 🛠️ Chcesz do nas dołączyć? (Dla studentów MFI)
Studiujesz informatykę na naszym Wydziale? Znasz HTML, CSS, JS albo lubisz bawić się we frameworkach i chciałbyś dorzucić swoją cegiełkę do naszych projektów? 

**Jesteśmy otwarci na współpracę!** Możesz śmiało przeglądać nasz kod, zgłaszać *Issues*, jeśli znajdziesz jakiegoś buga na naszych stronach, albo tworzyć *Pull Requesty* z nowymi funkcjonalnościami. Zawsze chętnie przygarniemy pomoc techniczną do Zespołów Roboczych RSS.
